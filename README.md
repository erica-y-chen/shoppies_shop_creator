# shoppies_shop_creator
*Shop Creator Infrastructure for Shoppies.co*

Shoppies Creator Sign Up Page

Build Instructions: 

On a Linux/Mac OS 

1. run `curl https://install.meteor.com/ | sh`
2. run `npm install`
3. run `npm start`

On a Windows 

1. run `choco install meteor`
2. run `npm install`
3. run `npm start`

App will run on `http://localhost:3000/`
